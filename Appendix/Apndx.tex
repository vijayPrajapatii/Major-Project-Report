\section{Metric Extraction Module}
This appendix presents a representative source listing from the pipeline implementation discussed in Chapter 4. The module shown below is \texttt{extract.py}, which is responsible for pulling deterministic performance counters out of a single raw log file and emitting them as a Pandas-friendly dictionary. Each regular expression in the \texttt{COUNTER\_PATTERNS} dictionary corresponds to one row of the parameter tables in Chapter 3 (Tables 3.2 and 3.3), so this module is the direct realization of the metric set documented earlier.

The function \texttt{extract\_metrics} reads the log file, applies every regular expression in turn, and stores the first numeric match for each counter. Counters that do not appear in a particular log are reported as \texttt{None} so that downstream comparison logic can distinguish ``not present'' from ``present and zero''. The returned dictionary is consumed by \texttt{compare.py}, which computes the percentage difference of every metric against the Cascade baseline using Equation 3.1 from Chapter 3.

"""extract.py - regex-driven metric extraction for one regression log."""

import re
from pathlib import Path

# Each entry maps a canonical metric name to a regex that captures
# its numeric value from the raw log file. The patterns are anchored
# to the surrounding banner text so the match is unambiguous.
COUNTER_PATTERNS = {
    "cycles":              r"Total cycles\s*[:=]\s*(\d+)",
    "IPS":                 r"IPS\s*[:=]\s*([\d.]+)",
    "bandwidth_MB":        r"Bandwidth\s*\(MB\)\s*[:=]\s*([\d.]+)",
    "Coproc_1_MAC":        r"COPROC_1_MAC\s*[:=]\s*(\d+)",
    "Coproc_1_CVT":        r"COPROC_1_CVT\s*[:=]\s*(\d+)",
    "Coproc_2_PKT":        r"COPROC_2_PKT\s*[:=]\s*(\d+)",
    "Coproc_1_STALLS":     r"COPROC_1_STALLS\s*[:=]\s*(\d+)",
    "Coproc_2_STALLS":     r"COPROC_2_STALLS\s*[:=]\s*(\d+)",
    "SCALAR":              r"SCALAR\s*[:=]\s*(\d+)",
    "spill_bytes":         r"spill_bytes\s*[:=]\s*(\d+)",
    "fill_bytes":          r"fill_bytes\s*[:=]\s*(\d+)",
    "selected_sequencer":  r"selected_sequencer\s*[:=]\s*(\S+)",
    "tiling_shape":        r"tiling_shape\s*[:=]\s*(\S+)",
    "hexnn_sha":           r"hexnn_sha\s*[:=]\s*([0-9a-f]+)",
}

NUMERIC_KEYS = {
    "cycles", "IPS", "bandwidth_MB",
    "Coproc_1_MAC", "Coproc_1_CVT", "Coproc_2_PKT",
    "Coproc_1_STALLS", "Coproc_2_STALLS",
    "SCALAR", "spill_bytes", "fill_bytes",
}


def extract_metrics(log_path: Path) -> dict:
    """Read one log file and return its metrics as a dictionary.

    Counters that do not appear in the log are returned as None so
    that downstream comparison can distinguish absent from zero.
    """
    text = Path(log_path).read_text(errors="ignore")
    out = {"graph": Path(log_path).stem}

    for name, pattern in COUNTER_PATTERNS.items():
        match = re.search(pattern, text)
        if match is None:
            out[name] = None
            continue
        value = match.group(1)
        if name in NUMERIC_KEYS:
            try:
                out[name] = float(value) if "." in value else int(value)
            except ValueError:
                out[name] = None
        else:
            out[name] = value
    return out


def extract_option(option_dir: Path) -> list:
    """Walk one option directory and extract metrics for every graph."""
    return [extract_metrics(p) for p in sorted(option_dir.glob("*.log"))]
